# wsb-piu-task-manager

Projekt semestralny z przedmiotu **Projektowanie interfejsów użytkownika** (WSB Merito, semestr 4).

Aplikacja do zarządzania zadaniami (Task Management App) budowana w React — z naciskiem na pełen proces projektowy UX/UI: od badania użytkownika, przez journey mapping, po implementację i testy.

## Status projektu

Projekt jest realizowany w 7 etapach:

1. **Etap 1 — Analiza UX** — ukończony
2. **Etap 2 — Journey Maps** — w trakcie (persony: Kasia, Marek, Anna)
3. Etap 3 — Information Architecture
4. Etap 4 — Wireframes
5. Etap 5 — Prototyp (high-fidelity)
6. Etap 6 — Implementacja (React)
7. Etap 7 — Testy użyteczności

## Struktura repo

- `Projetkowanie_projekt.pdf` — brief projektowy od prowadzącego
- `Etap1_Analiza_UX.md` — analiza UX, persony, problemy użytkownika
- `Etap2_Journey_Maps.md` — mapy podróży użytkownika

Kolejne etapy będą dodawane w ramach rozwoju projektu.

## Stack (planowany)

- TypeScript
- (do ustalenia na etapie 6 — state management, styling)
