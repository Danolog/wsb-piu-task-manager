# wsb-piu-task-manager

# Aplikacja do zarządzania zadaniami (Task Manager)

Projekt semestralny z przedmiotu **Projektowanie interfejsów użytkownika** (WSB Merito, semestr 4).
Aplikacja realizuje pełny proces projektowy UX/UI: od analizy potrzeb, przez makiety i prototyp, aż po implementację frontendową i testy użyteczności.

## Status projektu (wg etapów z briefu)

|---|------|----------------|
| 1 | Wprowadzenie i analiza potrzeb (Persony) | ✅ Zrobione     |

| 7 | Prezentacja i dokumentacja PDF | ✅ Zrobione |

## Uruchomienie lokalne

Aby uruchomić aplikację na swoim komputerze, upewnij się, że masz zainstalowane środowisko Node.js, a następnie wpisz w terminalu:

```bash
npm install
npm run dev
```
